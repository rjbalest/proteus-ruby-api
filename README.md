# proteus-ruby-api
Ruby binding to Bluecat Proteus SOAP API

## Dependencies

* [savon](http://savonrb.com) Version 2
* [rubytree](https://github.com/evolve75/RubyTree)

## Configuration
```
Bluecat::Api::WsdlUrl = 'http://localhost:3000/Services/API?wsdl'
```

## Usage

### POC demonstrations
```
ruby bluecat_poc.rb
```

### JSON Tree dump
```
ruby bluecat_tree.rb
```
